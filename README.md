# pts-plan

## Установка

Установить Node JS версии 16 и больше, затем выполнить команду:

Режим разработки:

```bash
npm run dev
```

Собрать проект:

```bash
npm run build
```

## О расширении

Целью расширения является улучшение функционала Битрикс24 CRM ресурса [plan.pixelplus.ru](https://plan.pixelplus.ru) для
работников компании.
После установки расширения для полноценной работы рекомендуется в настройках расширения указать Имя/Фамилию
пользователя, а также его уникальный ID в CRM. Далее отметить чекбоксами необходимый функционал и сохранить настройки.
При добавлении новых возможностей в расширении они будут по умолчанию деактивированы.

## Основные возможности расширения

* Редизайн сообщений
* Убирает панель с днями рождениями справа, чтобы не занимала лишнее место в канбане
* Возвращает онлайн статусы пользователей в чатах
* Закрывает уведомления, в которых не упоминается ваше имя
* В канбане добавляет кнопки редактирования задачи
* На странице проекта добавляет название проекта в title
* В результатах поиска убирает мусорные результаты на страницы форума
* Кнопка добавления тега текущего месяца к задаче в канбане
* Кнопка для разворачивания всех комментариев задачи
* Автоматический выбор единственного найденного сотрудника при поиске через "+"
* Кнопка для восстановления длинных ссылок в описании задачи и комментариях
* Выделение сотрудников, которые упоминаются в задаче, но которые не добавлены в неё
* На странице уведомлений выделяются сообщения и задачи, в которых упоминается ваше Имя Фамилия
* Кнопка открытия всех задач на странице [/alert](https://plan.pixelplus.ru/alert/)
* Жёлтым значком отмечаются сообщения, у которых в первых 100 символов сообщения не упоминается ни один пользователь или
  упоминается только автор сообщения (никому не адресованные сообщения)
* Кнопка удаления всех уведомлений данной задачи
* Скрытие изображений у задач в канбане
* Выделение в задачах TAGALL другим цветом
* Функционал для быстрой фильтрации уведомлений по группам на странице [/alert](https://plan.pixelplus.ru/alert/)
